## Критерий завершения этапа документации

Этап считается закрытым, когда:

- все документы из раздела “Критично” написаны и утверждены;
- у каждого документа есть владелец и дата последнего обновления;
- ссылки на документы добавлены в `README.md` и/или `docs/DOCS_INDEX.md`;
- документы используются в реальных релизах/инцидентах, а не только существуют формально.
